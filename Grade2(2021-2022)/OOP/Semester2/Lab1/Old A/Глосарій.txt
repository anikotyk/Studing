Нотатка - файл із текстом котрий ввів користувач. Має такі властивості як теги, архивована або не архивована, дата створення.
Меню - місце де відображаються усі нотатки створені користувачем. Також є поле для пошуку по тегам, пошуку в архіві, та створення нових нотаток чи керування вже створеними.
Редактор нотаток - вікно з текстом обраної\створеної нотатки, в якому її можно змінювати.
Тег - слово або кілька слів которі описують контент нотатки і використовуються для швидкого пошуку нотаток.
Архів - набір нотаток, котрі користувач відділив від інших, наприклад через те, що вони більше не потрібні.
Json Manager - класс для управління json файлом котрий зберігає інформацію про всі нотатки (дату останньої зміни, теги, архивована чи не архивована).
