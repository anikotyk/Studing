Меню - вікно программи у котрому відображаються усі нотатки створені користувачем. Також є поле для пошуку по тегам, пошуку в архіві, та створення нових нотаток чи керування вже створеними.
Редактор нотаток - вікно з текстом обраної\створеної нотатки. Є випадаючий лист функцій для додавання тегів, перенесення в архів\ з архіву, та видалення нотатки.
Тег - слово або кілька слів которі описують контент нотатки і використовуються для швидкого пошуку нотаток.
Архів - набір нотаток, котрі користувач відділив від інших, наприклад через те, що вони більше не потрібні.
Json Manager - класс для управління json файлом котрий зберігає інформацію про всі нотатки (дату останньої зміни, теги, скорочений текст нотатки).
MenuWindow - класс для управління вікном меню\пошуку нотаток.
TagsList - класс для додавання та видалення тегів.
MainWindow - класс для управління вікном редактору нотаток\ їх збереженням.
